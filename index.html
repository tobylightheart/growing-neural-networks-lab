Companion project

Growing Neural Networks Lab

Small, inspectable implementations and static experiments for constructive, growing, pruning, and plastic neural-network mechanisms.

Remit

The main garden owns papers, reviews, synthesis, and historical context. This lab owns runnable mechanisms: pure-Python scripts, tiny traces, static demos, and tests.

Available lab

Cascade-Correlation on XOR

A minimal demonstration of growing one residual-correlated hidden feature to solve XOR.

Design principles

  • No heavy ML frameworks by default.
  • Each lab should run locally and be readable in one sitting.
  • Claims stay modest unless tests and sources support more.
  • Link back to the literature garden instead of duplicating reviews.
