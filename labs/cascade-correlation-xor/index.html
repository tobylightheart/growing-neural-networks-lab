Toy mechanism trace

Cascade-Correlation on XOR

A minimal demonstration of adding one residual-correlated hidden feature, freezing it, and retraining the output layer.

Scope: this is a small educational mechanism, not a full reproduction of Fahlman & Lebiere (1990). For paper context, use the main Growing Neural Networks literature garden.

1. Baseline

A linear model sees only bias, x₁, and x₂. It cannot separate XOR.


      

      

        
2. Candidate unit

        
A tiny search finds a sigmoid feature whose activation correlates with the residual error.

        

      

      

        
3. Grown model

        
The hidden feature is frozen. The output model is refit using bias, inputs, and the new feature.
